# 🌌 NexusOS (Project Nexus)

> **The Sovereign AI Operating System Shell**  
> *Autonomous. Local-First. Self-Healing.*

NexusOS is a High-Fidelity AI Operating Environment designed for researchers, artists, and developers. Built on a hybrid architecture of Electron, Node.js (ESM), and Python, it integrates a sophisticated agentic AI loop (Nexus-Prime) with a premium Glassmorphism design system.

---

## 🚀 Core Pillars

### 1. 🧠 Nexus-Prime Engine
A fully autonomous, local-only agentic loop powered by LM Studio. 
- **Smart Model Routing:** Intelligent switching between 7B (UI tasks) and 14B (Complex logic).
- **VRAM Control:** Automated model loading, unloading, and OOM fallback.
- **Surgical Patching:** SEARCH/REPLACE atomic code modifications.

### 2. 🛡️ Nexus-Sentinel (V3)
The system's immune system featuring Thermal Sentinel innovations. An autonomous self-healing service that intercepts crashes in React, Node, and Python runtime.
- **Context Sniping:** 80-line granular error extraction.
- **Terminator Loop:** Recursive self-correction of syntax and logic errors.
- **IPC Healing:** Auto-patches missing IPC registrations.

### 3. 🎨 Premium Glassmorphism
A state-of-the-art UI shell built with Tailwind CSS v4.
- **HUD Aesthetics:** Transparent panels, ambient noise filters, and glowing borders.
- **Zero-Tolerance CSP:** Hardened Content Security Policy for maximum privacy.

### 4. 🧬 Animus & Architect
- **Animus:** Legacy code DNA sequencer that evolves old scripts into modern Nexus modules.
- **Architect:** Proactive feature invention daemon that proposes and builds system improvements.

## 🛠️ Tech Stack

- **Shell:** Electron v29.x
- **Frontend:** React + Vite + Tailwind CSS v4 + Framer Motion
- **Backend:** Node.js ESM (apps/backend-core)
- **Intelligence:** LM Studio (Local) + Google Gemini (Hybrid Cloud)
- **Database:** SQLite (Knowledge & Telemetry)
- **Daemons:** Python 3.10+ (Core Logic & Tooling)

## ⌨️ Developer Setup

### Prerequisites
- Node.js 20+
- pnpm 8+
- Python 3.10 (Add to PATH)
- LM Studio (Running on port 1234/1235)

### Installation
```bash
# Clone the repository
git clone https://github.com/your-repo/NexusOS.git
cd NexusOS

# Install dependencies
pnpm install
```

### Execution
Use the provided launchers in the `launchers/` directory:
- `start-app.bat`: Full deep-clean and native app launch.
- `start-web.bat`: Vite Dev Server for UI development.
